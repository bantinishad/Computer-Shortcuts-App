# Computer Shortcuts 

Computer Shortcuts is a modern Android application built using **Kotlin** and **XML** that helps users quickly learn and search keyboard shortcuts for different operating systems like **Windows**, **Mac**, and **Linux**.

The app provides categorized shortcuts, search functionality, favorites support, and a clean user interface for better productivity learning.

---

## Features

-  Learn Computer Keyboard Shortcuts
-  Support for Windows, Mac & Linux
-  Search Shortcuts Instantly
-  Category Wise Filtering
- Add Shortcuts to Favorites
-  Detailed Shortcut Information
-  Clean and User-Friendly UI
- Fast JSON-Based Data Loading
-  Favorites Saved Using SharedPreferences

---

##  Features Overview

###  Keyboard Shortcuts

The app provides shortcuts for:
- Windows
- Mac
- Linux

### Smart Search

Users can search shortcuts using:
- Shortcut Action
- Description
- Shortcut Keys

###  Category Filter

Filter shortcuts category-wise for easier learning.

###  Favorites Support

Users can save important shortcuts into favorites for quick access later.
